# คู่มือผู้ใช้งาน E-Memo ครั้งแรก

**บริษัท ไทยซอสเซส มาร์เก็ตติ้ง จำกัด**

---

## E-Memo คืออะไร?

E-Memo เป็นระบบส่งและอนุมัติ Memo ออนไลน์ ใช้แทนการส่ง Memo กระดาษหรืออีเมลทั่วไป โดยมีขั้นตอนอนุมัติหลายระดับ ลายเซ็นดิจิทัล และการแจ้งเตือนผู้เกี่ยวข้อง

**เข้าใช้งานที่:** [https://ememo-thaisausagemarketing.vercel.app](https://ememo-thaisausagemarketing.vercel.app)

---

## ก่อนเริ่มใช้งาน

Admin จะสร้างบัญชีให้คุณและส่งอีเมลจาก **noreply.ememo@tgm.co.th** ซึ่งมีข้อมูลดังนี้:

| รายการ | ตัวอย่าง |
|--------|----------|
| Username | `somchai` |
| Password ชั่วคราว | `Abc12345` |
| ลิงก์ตั้งรหัสผ่าน | ปุ่มในอีเมล |

> **สำคัญ:** ตรวจสอบกล่อง **Inbox, Spam และ Junk** หากยังไม่เห็นอีเมล

---

## ขั้นตอนที่ 1 — ตั้งรหัสผ่าน (เลือกวิธีใดวิธีหนึ่ง)

### วิธี A: ใช้ลิงก์ในอีเมล (แนะนำ)

1. เปิดอีเมลจาก `noreply.ememo@tgm.co.th`
2. กดปุ่ม **ตั้งรหัสผ่าน** หรือคัดลอกลิงก์ไปเปิดในเบราว์เซอร์
3. ระบบจะแสดงหน้า **ตั้งรหัสผ่านใหม่**
4. กรอกรหัสผ่านใหม่ (อย่างน้อย 6 ตัวอักษร) และยืนยันอีกครั้ง
5. กด **บันทึกรหัสผ่านใหม่**
6. กด **กลับหน้า Login** แล้วเข้าสู่ระบบด้วยรหัสผ่านใหม่

> ลิงก์มีอายุ **1 ชั่วโมง** หากหมดอายุ ให้ใช้วิธี B หรือขอ Admin ส่งอีเมลใหม่

### วิธี B: เข้าสู่ระบบด้วยรหัสชั่วคราว

1. เปิด [https://ememo-thaisausagemarketing.vercel.app](https://ememo-thaisausagemarketing.vercel.app)
2. กรอก **Username** หรือ **Email** และ **Password ชั่วคราว** จากอีเมล
3. กด **เข้าสู่ระบบ**
4. ระบบจะบังคับให้เปลี่ยนรหัสผ่านทันที (หน้า **ตั้งรหัสผ่านใหม่**)
5. ตั้งรหัสผ่านใหม่แล้วกด **บันทึกและเข้าใช้งาน**
6. ระบบจะเปิด **ทัวร์แนะนำ E-Memo** อัตโนมัติ — ตามขั้นตอนเพื่อทำความเข้าใจเมนูหลักทั้งระบบ

---

## ทัวร์แนะนำหลัง Login ครั้งแรก

หลังตั้งรหัสผ่านสำเร็จ ระบบจะพาทัวร์เมนูสำคัญโดยอัตโนมัติ:

| ลำดับ | หัวข้อ |
|-------|--------|
| 1 | ยินดีต้อนรับ |
| 2 | ปุ่มสร้าง Memo |
| 3 | ภาพรวม |
| 4 | กล่องขาเข้า |
| 5 | Memo ของฉัน |
| 6 | ค้นหา |
| 7 | Route อนุมัติ |
| 8 | โปรไฟล์ / ลายเซ็น |
| 9 | พร้อมใช้งาน |

- กด **ถัดไป** เพื่อไปขั้นถัดไป หรือ **ข้ามทัวร์** หากคุ้นเคยแล้ว
- Admin / Super Admin จะเห็นขั้นตอนเพิ่มเติมตามสิทธิ์ (Memo ทั้งหมด, จัดการ User, ตั้งค่าระบบ)

---

## ขั้นตอนที่ 2 — เข้าสู่ระบบ

1. เปิดเว็บ E-Memo
2. กรอกข้อมูลในช่อง Login:

| ช่อง | วิธีกรอก |
|------|----------|
| Username หรือ Email | กรอก `username` เช่น `somchai` **หรือ** อีเมลเต็ม เช่น `somchai@tgm.co.th` |
| Password | รหัสผ่านที่คุณตั้งไว้แล้ว |

3. กด **เข้าสู่ระบบ**

> หากกรอกแค่ Username (ไม่มี @) ระบบจะเติม `@tgm.co.th` ให้อัตโนมัติ

---

## ขั้นตอนที่ 3 — ตั้งลายเซ็น (แนะนำทำก่อนอนุมัติ Memo)

ลายเซ็นใช้ตอนอนุมัติ Memo ควรตั้งไว้ก่อนใช้งานจริง

1. คลิกที่ **ชื่อ/รูปโปรไฟล์** มุมล่างซ้าย (Desktop) หรือไอคอนโปรไฟล์ (มือถือ)
2. ในหน้าโปรไฟล์ เลือก **ลายเซ็น**
3. **วาดลายเซ็น** ในกล่อง หรือ **อัปโหลดรูปภาพ** ลายเซ็น
4. กด **บันทึก**

---

## ภาพรวมหน้าจอหลัก

หลัง Login สำเร็จ จะเห็เมนูด้านซ้าย (หรือแถบด้านล่างบนมือถือ):

| เมนู | ใช้ทำอะไร |
|------|-----------|
| **ภาพรวม** | ดูสรุป Memo และงานที่เกี่ยวข้อง |
| **กล่องขาเข้า** | Memo ที่รอคุณอนุมัติ (มีตัวเลขแดงแจ้งจำนวน) |
| **Memo ของฉัน** | Memo ที่คุณสร้างหรือเป็นผู้ร่าง |
| **ค้นหา** | ค้นหา Memo ตามคำค้น |
| **Route อนุมัติ** | บันทึก workflow อนุมัติที่ใช้บ่อย เพื่อโหลดซ้ำได้เร็ว |

---

## การสร้าง Memo ใหม่

1. กด **+ สร้าง Memo ใหม่**
2. กรอก **หัวข้อ** และ **เนื้อหา**
3. เลือก **หมวดหมู่**
4. ตั้ง **ขั้นตอนการอนุมัติ (Workflow)** — เลือกผู้อนุมัติแต่ละระดับ
   - หรือโหลดจาก **Route อนุมัติ** ที่บันทึกไว้
5. (ถ้าต้องการ) เลือก **ผู้รับแจ้งเตือน** ทางอีเมล
6. (ถ้าต้องการ) แนบไฟล์
7. เลือก:
   - **บันทึกร่าง** — เก็บไว้แก้ไขก่อนส่ง
   - **ส่งเพื่ออนุมัติ** — ส่งเข้าขั้นตอนอนุมัติทันที

---

## การอนุมัติ Memo

1. ไปที่ **กล่องขาเข้า**
2. คลิก Memo ที่ต้องการพิจารณา
3. อ่านรายละเอียด แล้วเลือก:
   - **อนุมัติ** — ระบบจะใช้ลายเซ็นที่ตั้งไว้ (หรือให้วาด/อัปโหลดใหม่)
   - **ไม่อนุมัติ** — กรอกเหตุผล
4. Memo จะส่งต่อไปยังผู้อนุมัติระดับถัดไป หรือเสร็จสิ้นตาม workflow

---

## ลืมรหัสผ่าน

1. ที่หน้า Login กด **ลืมรหัสผ่าน / เข้าใช้ครั้งแรก?**
2. กรอก Username หรือ Email
3. กด **ส่งลิงก์ตั้งรหัสผ่าน**
4. เปิดอีเมลจาก `noreply.ememo@tgm.co.th` แล้วตั้งรหัสผ่านใหม่
5. กลับมา Login ด้วยรหัสผ่านใหม่

## คำถามที่พบบ่อย (FAQ)

### เปิดลิงก์ในอีเมลแล้วเจอหน้า "Log in to Vercel"

ลิงก์อาจชี้ไป URL ทดสอบของระบบ ไม่ใช่ URL หลัก — **ติดต่อ Admin** ให้ส่งอีเมลใหม่ หรือเข้าที่ [https://ememo-thaisausagemarketing.vercel.app](https://ememo-thaisausagemarketing.vercel.app) โดยตรง

### ลิงก์ตั้งรหัสผ่านหมดอายุ

ลิงก์ใช้ได้ **1 ชั่วโมง** — ใช้เมนู **ลืมรหัสผ่าน** ที่หน้า Login เพื่อขอลิงก์ใหม่

### Login ไม่ได้ / Email หรือ Password ไม่ถูกต้อง

- ตรวจสอบว่าใช้ Username หรือ Email ที่ Admin ส่งให้
- ตรวจสอบ Caps Lock
- หากยังไม่ได้ตั้งรหัสผ่านใหม่ ให้ใช้ลิงก์จากอีเมลก่อน

### ไม่มีบัญชีในระบบ

ติดต่อ **Admin / Super Admin** ของ E-Memo ให้สร้างบัญชีและส่งอีเมลแจ้งข้อมูลเข้าใช้งาน

### อนุมัติ Memo แล้วไม่มีลายเซ็น

ไปที่ **โปรไฟล์ → ตั้งลายเซ็น** ก่อน แล้วลองอนุมัติใหม่

## สรุปขั้นตอนสำหรับผู้ใช้ใหม่

```
รับอีเมลจาก Admin
       ↓
ตั้งรหัสผ่าน (ลิงก์ในอีเมล หรือ Login แล้วเปลี่ยนรหัส)
       ↓
Login เข้า E-Memo
       ↓
ทัวร์แนะนำระบบ (อัตโนมัติครั้งแรก)
       ↓
ตั้งลายเซ็นในโปรไฟล์
       ↓
เริ่มใช้งาน — สร้าง Memo / อนุมัติ Memo ในกล่องขาเข้า
```
